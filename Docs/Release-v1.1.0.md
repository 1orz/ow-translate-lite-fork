# OW Translator Lite v1.1.0

这一版主要围绕性能、手动识别、回话输入、更新体验和稳定性做了改进。

## 更新要点

- 新增手动模式：如果担心自动 OCR 影响帧率，可以切到手动模式，只在需要时识别一次。
- 翻译框新增相机按钮：点击即可立即识别当前框选区域；自动模式下也能使用。
- 手动模式支持 `Ctrl+Shift+T` 快捷键，按一下就识别一次。
- 自动识别更省资源：空闲时降低 OCR 频率，看到聊天后再短时间加快识别。
- 回话翻译更稳：剪贴板被微信等软件占用时，会把译文留在输入框里，方便手动复制。
- 修复多屏和屏幕边缘框选导致的区域错误、连续报错问题。
- 优化浅色主题和 overlay 下拉菜单显示，文字更清楚。
- 修复 overlay 回话条语言选择按钮背景不随透明度变化、在浅色背景下显得突兀的问题。
- 新增更新后说明弹窗和一键卸载器。

## 新功能

- 新增“手动模式”：开启后不会自动截图、巡逻或 OCR，只在用户主动触发时识别一次。
- Overlay 回话条新增相机按钮：自动模式和手动模式下都可以点击，立即对当前框选区域执行一次 OCR burst。
- 手动模式支持快捷键 `Ctrl+Shift+T`，可快速识别一次当前聊天区域。
- 新增“更新后首次打开”的本次更新说明弹窗，帮助用户了解新功能和注意事项。
- 发布包包含一键卸载器，可删除便携包目录和 `%AppData%\OWTranslatorLite` 用户数据。

## 性能与 OCR

- 自动识别逻辑改为低频 OCR 探测 + 活跃 burst：
  - 空闲时降低 OCR 频率，减少 CPU 开销。
  - 识别到聊天后进入短时间活跃窗口，提高新消息响应速度。
- 相机按钮触发的是 3 帧短 burst，兼容现有多帧确认去重逻辑，减少“点了但没翻译”的情况。
- OCR 调用增加互斥保护，避免自动 OCR 和手动触发同时运行。

## 回话输入与剪贴板

- 回话翻译增加“自动复制回话译文”开关。
- 关闭自动复制时，译文会留在回话输入框中，用户可手动复制。
- 自动复制失败时不再直接判定回话翻译失败，而是显示译文并提示手动复制。
- 回话输入框新增复制按钮，复制时使用重试逻辑，缓解微信等软件占用剪贴板导致的失败。

## 稳定性修复

- 修复多屏、副屏、屏幕边缘框选时区域坐标不稳定的问题。
- 截图前会裁剪并校验虚拟屏幕范围，避免区域越界后连续刷错。
- 区域无效时自动暂停识别，并只记录明确错误信息。
- 更新流程继续保留 `%AppData%\OWTranslatorLite` 中的设置、API Key、日志和诊断数据。

## 界面与体验

- 主窗口新增“识别模式”选项，并提供即时显示的问号说明。
- 调整“使用说明”“外观”“回话输入”等设置区块顺序。
- 修复浅色主题下 overlay 语言下拉文字看不清的问题。
- 统一 overlay 回话条语言选择、输入框、相机按钮和复制按钮的深灰透明背景。
- 更新应用图标为透明底新版图标。
- 更新弹窗和发布包现在会携带本地更新说明，更新后首次打开可直接查看要点。

## 诊断与开发

- Lab 工具增加 OCR gate 评估能力，用于对真实截图样本分析误判和漏判。
- 性能和门控评估保留在 Lab，不污染正式主界面。

## 更新方式

自动更新会下载最新 `OWTranslatorLite-v1.1.0-portable-win-x64.zip`，校验发布包提供的 sha256 后替换外层目录中的 `app/`。更新时会保留最近一次旧 `app` 备份在 `.update-backup/`，并保留 `%AppData%\OWTranslatorLite` 中的设置、API Key、日志和诊断文件。

如果自动下载失败，可以手动从 GitHub Releases 下载最新 zip，放到外层 `OWTranslatorLite/` 文件夹，再运行 `OWTranslatorLiteUpdater.exe` 一键更新。

如需卸载，运行外层 `OWTranslatorLiteUninstall.exe`。卸载器会删除当前便携包目录和 `%AppData%\OWTranslatorLite`，包括 API Key、设置、日志、诊断包和 overlay 历史。
